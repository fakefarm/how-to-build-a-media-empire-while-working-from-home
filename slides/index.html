Terminal & Text Editor

Install

					$ gem install middleman
					$ middleman init demo --template=blog
					$ bundle exec middleman # start server at http://localhost:4567/

Deploy

					
# GEMFILE
gem "middleman-deploy" # deploy on github
					
					
# config.rb
activate :deploy do |deploy|
	deploy.build_before = true
	deploy.method = :git
end
					
					
(create a repo on github)

					$ bundle exec middleman deploy # deploys to gh-pages
